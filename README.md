# Iris Data Visualization

## Objective

The objective of this project is to visualize patterns, distributions, and relationships in the Iris dataset using different types of graphical representations.

## Dataset

Dataset: Iris Dataset

File Used: Iris.csv

Number of Rows: 150

Number of Columns: 6

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* VS Code
* Jupyter Notebook

## Steps Performed

### 1. Data Loading

The Iris dataset was loaded using Pandas.

### 2. Data Inspection

Dataset structure, shape, and basic information were examined.

### 3. Data Validation

Missing values and dataset consistency were checked.

### 4. Bar Chart Visualization

A bar chart was created to display the number of samples for each Iris species.

### 5. Histogram Visualization

A histogram was generated to show the distribution of Sepal Length values.

### 6. Scatter Plot Visualization

A scatter plot was created to visualize the relationship between Sepal Length and Petal Length for different species.

### 7. Feature Comparison

Average feature values of Sepal Length, Sepal Width, Petal Length, and Petal Width were compared across species using a bar chart.

## Visualizations Created

* Bar Chart
* Histogram
* Scatter Plot
* Feature Comparison Chart

## Results

* Species distribution was visualized successfully.
* Feature distributions were clearly observed.
* Relationships between flower measurements were identified.
* Differences among Setosa, Versicolor, and Virginica species were highlighted through visual analysis.
